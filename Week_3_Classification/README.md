## 📌 Executive Problem Statement

**HELP International**, an international humanitarian NGO, has raised **$10 million USD** to deploy strategic aid to countries in direst need. 

As the Lead Data Scientist, the goal of this project is to build an **End-to-End ML Pipeline** that uses data-driven country segmentation to identify and rank countries most in need of aid. Additionally, we built a predictive intelligence model capable of classifying any new country's development tier automatically.

## 🧠 MLOps & Machine Learning Lifecycle

This repository strictly follows industry-standard MLOps practices, including Custom Exception Handling and Logging modules. The pipeline execution follows:

1. **Exploratory Data Analysis (EDA):** Correlation heatmaps, feature distributions, and bivariate analysis.
2. **Data Scaling & Preprocessing:** Standardisation of socio-economic features.
3. **Dimensionality Reduction:** Principal Component Analysis (PCA) for 2D visual projections.
4. **Unsupervised Clustering:** 
   - **K-Means (K=3):** Validated using the Elbow Method and Silhouette Analysis.
   - **DBSCAN:** Density-based spatial clustering to detect geopolitical outliers (Noise).
5. **Ensemble Classification (Semi-Supervised):** 
   - Using the clusters generated by K-Means as the ground truth labels to train supervised classifiers.
   - **Random Forest Classifier**
   - **XGBoost Classifier**
6. **Hyperparameter Optimisation:** GridSearchCV with Stratified 5-Fold Cross-Validation.
7. **Actionable Intelligence:** Formulating a composite **Needs Index** to rank the top 15 countries requiring immediate humanitarian aid.

## 📊 Dataset Features

The dataset comprises 167 countries evaluated across 9 socio-economic and health indicators:

| Feature | Description |
|---|---|
| `child_mort` | Child mortality (under 5) per 1,000 live births |
| `exports` | Exports as % of GDP per capita |
| `health` | Total health spending as % of GDP per capita |
| `imports` | Imports as % of GDP per capita |
| `income` | Net income per person (USD) |
| `inflation` | Annual GDP growth rate (%) |
| `life_expec` | Average life expectancy (years) |
| `total_fer` | Total fertility rate |
| `gdpp` | GDP per capita (USD) |

## 🏆 Final Results & Model Performance

After Hyperparameter tuning, the **XGBoost Classifier** achieved the highest generalisation performance.

| Model | Accuracy | F1-Macro |
|---|---|---|
| **XGBoost (Tuned)** | **0.97** | **0.97** |
| Random Forest (Tuned) | 0.94 | 0.94 |
| XGBoost (Baseline) | 0.94 | 0.93 |
| Random Forest (Baseline) | 0.91 | 0.89 |
